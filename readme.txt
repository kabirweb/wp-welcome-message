== Description ==
WP Welcome Message is a wordpress plugin, which help your to make any announcement, special events, special offer, signup message or such kind of message, displayed upon your website's visitors when the page is load through a popup box.

== Installation ==
1. Upload `wp-welcome-message` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
